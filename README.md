## 📖 ဆော့ဖ်ဝဲလ် အကြောင်း (About the Software)
**ဝေါဟာရဒီပံ (Wawhara Dipan)** သည် လေ့လာသင်ယူသူများနှင့် အများပြည်သူ ဗဟုသုတတိုးတက်ရေးအတွက် ရည်ရွယ်၍ အခမဲ့ ဖြန့်ချိသော Windows Desktop သုံး Myanmar-English / English-Myanmar Dictionary ဆော့ဖ်ဝဲလ် ဖြစ်ပါသည်။

**Wawhara Dipan** is a free offline Myanmar-English and English-Myanmar Dictionary application designed for Windows Desktop. It aims to assist students, language learners, and the general public in enhancing their vocabulary and language skills.

---

## ✨ အဓိက ပါဝင်သော စနစ်များ (Key Features)

* **🔍 နှစ်ဘာသာ ရှာဖွေနိုင်ခြင်း (Bi-directional Search)**
  * English ➔ Myanmar ရော Myanmar ➔ English ပါ အလွယ်တကူ ရှာဖွေနိုင်ခြင်း။ (*Search words effortlessly both ways.*)
* **🔊 အသံထွက်စနစ် (Audio Pronunciation)**
  * အင်္ဂလိပ်စာလုံးများ၏ အသံထွက်ကို 🔊 ခလုတ်နှိပ်၍ မှန်ကန်စွာ နားထောင်နိုင်ခြင်း။ (*Built-in text-to-speech feature to listen to correct English pronunciations.*)
* **⭐ မှတ်သားရန်စနစ် (Bookmark / Favorite System)**
  * မိမိအလွတ်ကျက်ချင်သော သို့မဟုတ် ခဏခဏကြည့်လိုသော စာလုံးများကို ⭐ ကြယ်ပွင့်နှိပ်၍ သီးသန့် သိမ်းဆည်းထားနိုင်ခြင်း။ (*Save and organize difficult or important words for future review.*)
* **📋 ကူးယူခြင်းစနစ် (Easy Copy Text)**
  * စာလုံးနှင့် အဓိပ္ပာယ်ရှင်းလင်းချက်များကို clipboard ထဲသို့ 📋 တစ်ချက်နှိပ်ရုံဖြင့် အလွယ်တကူ Copy ကူးယူနိုင်ခြင်း။ (*Quickly copy definitions and meanings to your clipboard.*)

## 📥 ဒေါင်းလုဒ်နှင့် ထည့်သွင်းနည်း (Download & Installation)
1. Download the installer (`WawharaDipan_Setup_v1.0.1.zip` or `.exe`) from the [Releases](https://github.com/YOUR_GITHUB_USERNAME/wawharadipan.github.io/releases) page.
2. Run the installer and follow the setup wizard.
3. Complete the installation and launch from Start Menu or Desktop shortcut.

> 💡 **First Run Note:** Database will be automatically extracted to `%APPDATA%\WawharaDipan`. Favorites and user data are stored separately, so uninstalling will not delete your saved favorites.
